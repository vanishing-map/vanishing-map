visual contract 굵기는 선택한 필터의 유량, 색상은 특성 레이어 예약.
전국 화면은 시간축 슬라이더로 선택한 월의 interregional OD만 사용한다.
공공데이터 API 안내 기준 조회 가능 시작월이 2022년 10월이라, 현재 파일은 2022년 10월부터 2026년 5월까지 44개월이다.
같은 시도 내부 이동은 수치가 커서 전국 arc 해석을 압도하므로 제외했다.
연령과 성별 필터는 행정안전부 나이별 남녀 이동 집계에서 다시 만든 wide CSV를 기준으로 계산한다.
